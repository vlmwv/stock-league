주식 예측 리그
1. 설명
  - 매일 5개의 종목을 선정하여 유저들이 상승/하락을 예측하는 게임
  - 매일 08:00~20:00까지는 10/30/50분마다 전자공시/뉴스 수집
  - 매일 20:20에 시세 확인/저장, 정답자 확인, 랭킹 집계 
  - 매일 20:40에 KRX 상위 100종목을 가져와서 DB에 저장
  - 매일 21:00에 결과 노출
  - 매일 21:20에 5개 종목 선정, LLM으로 요약
  - 매일 22:00에 5개 종목에 대한 예측 시작
  - 매일 08:00에 예측 마감

2. 기술 스펙
  - nuxt.js, vercel, 슈파베이스, redis
  - vue, typescript, tailwindcss 
  - docker-compose 활용


2. 기능 상세설명_
  - 회원 가입은 이메일 가입, OAuth2 소셜 로그인 지원, JWT 토큰 발급
  - api 구분은 사용자, 관리자, 공통, 외부로 구분되면, api path와 권한이 분리됨
  - 게임 참여 종목 노출시 LLM 요약정보를 같이 노출함
  - 게임 5개 종목은 모든 사용자가 동일함 
  - 예측 선택은 상승은 위로 플릭, 하락은 아래로 플릭, 롱클릭은 찜하기
  - 찜한 종목은 찜한 목록에서 확인 가능
  - 찜한 종목이 게임 노출시 하트 표시
  - 시세 확인 저장시 당일 시세, 변동금액, 비율을 이력으로 저장
  - 정답확인시 승률,최다접자,정답률 랭킹도 집계 
    + 주간랭킹,월간랭킹,연간랭킹,역대랭킹으로 집계
    + 기본랭킹은 월간 승률이며, 승률은 규정타석 매일 3회 평균이상만 포함하며, 총 100위까지 노출됨 
    + 매주일요일,매주첫째주일요일,매년1월1일 리셋
  - 종목 저장시 기존종목은 시가총액 순위만 업데이트
  - 결과 노출은 팝업으로 본인의 예측결과와 랭킹 상승 숫자를 드라마틱하게 보여줌
  - 5개 종목 선정시 LLM을 통하여 종목 요약정보를 저장함
  - 전자공시/뉴스 수집시 LLM을 통하여 종목 요약정보를 저장함
  - 요약정보가 수집된 종목중에 5종목을 선별하여 추천 종목을 보여줌
  - 추천종목은 좌우로 이동시키면서 새로운 종목을 볼 수 있으며, 롱클릭은 찜하기

3. 사용자 화면
  - 메인 화면, 게임참여, 찜한 목록, 추천 종목, 랭킹, 마이페이지 등등
